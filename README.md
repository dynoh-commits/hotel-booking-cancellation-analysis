## Project Goal
Predict hotel reservation cancellations and analyze customer behavior patterns.

## Dataset
Hotel booking demand dataset

## Project Structure
- preprocessing/
- modeling/
- results/

## Models
- Logistic Regression
- Random Forest
- KMeans Clustering

## Evaluation
- Stratified 5-Fold Cross Validation
- F1-score
- Precision / Recall
